Description

This is an event which tests coding skills,focus, memory of an individual.
 Level 1: what to do? A group of pictures are shown to the participant. The picture includes keywords,evolving technologies, etc. The participant need to remember the order of the pictures and answer accordingly.
Level 2: A language can be selected based on their preference languages C , C++. The glance of the code is about 3-5 minutes and the participate need to code without using backspace, arrows, control in the given time.
Level 3: It is similar to level 2. The standards of the programs are increased and the time limit is set accordingly.
